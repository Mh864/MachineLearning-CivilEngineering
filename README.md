How to Run: 
```bash 
pip install -r requirements.txt 
python run_api.py 
# Terminal 1 — starts API at localhost:8000 
cd frontend1 CC npm install CC npm run dev # Terminal 2 — dashboard at localhost:3000 
docker compose up
``` 

The Final Report and slides are present in the Rapport&Powerpoint folder.
The docs folder contain more information on each part of the project.
